RELATIONAL DATA GENERATOR
#######################################

A utilitiy to create realistic and scaled data.

The purpose of this utility is to assist with the
creation of Business Intelliegence Visualizations.

Data can be generated by defining relationships
between pre-existing columns and choosing the
appropriate distributions to represent the result
data.

The end creation is intended to fill the schema of
a relational database with correct and accurate
data. This schema can then be used to create
realistic BI reporting.
